# Tidyverse Practice Examples

This repository contains practice examples using the **tidyverse** ecosystem in R.  
The goal is to explore data manipulation, transformation, and visualization using tidyverse tools such as:

- dplyr
- ggplot2
- tibble
- readr
- purrr

## Project Files

- **index.qmd** — main Quarto file with tidyverse examples  
- **index.html** — rendered output used for GitHub Pages  
- **_quarto.yml** — Quarto project configuration  
- **README.md** — documentation for this repository
